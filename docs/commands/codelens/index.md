# Code Lens Commands

**Description:** Inline code actions in ArgoCD YAML files for quick access to common operations.

## [ArgoCD Yaml Detection](./argocd-yaml-detection.md)

Automatically detect ArgoCD YAML files and provide inline code actions for quick application creation and management.

## [Create Application](./create-application.md)

Quickly create an ArgoCD application directly from a YAML code lens.

## [Create Template](./create-template.md)

Quickly create an application template directly from a YAML code lens.
