# ArgoCD Yaml Detection

**Description:** Automatically detect ArgoCD YAML files (Application and ApplicationSet resources) in the editor and display inline code lens indicators, providing quick access to common operations like creating applications or templates.

**Type:** Code Lens

**Keyboard Shortcut:** None

## Usage Flow

1. Open an ArgoCD YAML file in the editor
   ![](../../assets/commands/codelens/argocd-yaml-detection-step1.jpg)
2. Code lens indicators appear above Application and ApplicationSet resources
   ![](../../assets/commands/codelens/argocd-yaml-detection-step2.jpg)
3. Click on the code lens to view available actions
   ![](../../assets/commands/codelens/argocd-yaml-detection-step3.jpg)
