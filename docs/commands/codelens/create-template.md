# Create Template

**Description:** Quickly create and save an application template directly from an ArgoCD YAML file using a code lens, capturing the current configuration for reuse when creating future applications.

**Type:** Code Lens

**Keyboard Shortcut:** None

## Usage Flow

1. Open an ArgoCD YAML file with an Application resource
   ![](../../assets/commands/codelens/create-template-step1.jpg)
2. Click the code lens above the Application resource
   ![](../../assets/commands/codelens/create-template-step2.jpg)
3. Select "Create Template" from the available actions
   ![](../../assets/commands/codelens/create-template-step3.jpg)
4. Enter template details and confirm
   ![](../../assets/commands/codelens/create-template-step4.jpg)
