# Create Application

**Description:** Quickly create a new ArgoCD application directly from a YAML code lens inline in an ArgoCD or Application YAML file, streamlining application creation without leaving the editor.

**Type:** Code Lens

**Keyboard Shortcut:** None

## Usage Flow

1. Open an ArgoCD YAML file with an Application resource
   ![](../../assets/commands/codelens/create-application-step1.jpg)
2. Click the code lens above the Application resource
   ![](../../assets/commands/codelens/create-application-step2.jpg)
3. Select "Create Application" from the available actions
   ![](../../assets/commands/codelens/create-application-step3.jpg)
4. Confirm and wait for application creation
   ![](../../assets/commands/codelens/create-application-step4.jpg)
