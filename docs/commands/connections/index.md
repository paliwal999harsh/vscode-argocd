# Connections Commands

**Required Permissions:** ArgoCD connection management access

## [Add Connection](./add-connection.md)

Add a new ArgoCD server connection to manage multiple ArgoCD instances.

## [List Connections](./list-connections.md)

View all configured ArgoCD server connections.

## [Switch Connection](./switch-connection.md)

Switch to a different ArgoCD server connection.

## [Edit Connection](./edit-connection.md)

Modify the details of an existing ArgoCD server connection.

## [Delete Connection](./delete-connection.md)

Delete an ArgoCD server connection.
