# Repository Commands

**Required Permissions:** ArgoCD repository read/write access

## [Add Repository](./add-repository.md)

Add a Git/Helm/OCI repository to ArgoCD and show in Repository view.

## [Copy Repository URL](./copy-repository-url.md)

Copy the URL of a Git/Helm/OCI repository to clipboard.

## [Create Application from Repository](./create-application-from-repository.md)

Create a new ArgoCD application from an existing Git/Helm/OCI repository.

## [Refresh Repositories](./refresh-repositories.md)

Refresh the status and information of ArgoCD repositories.

## [Delete Repository](./delete-repository.md)

Delete a Git/Helm/OCI repository from ArgoCD.
