# Template Commands

**Required Permissions:** ArgoCD template read/write access

## [Add Template](./add-template.md)

Create and save a new application template for quick application creation.

## [Edit Template](./edit-template.md)

Modify an existing application template.

## [Create Application from Template](./create-application-from-template.md)

Create a new ArgoCD application using a saved template.

## [Delete Template](./delete-template.md)

Delete a saved application template.

## [Refresh Templates](./refresh-templates.md)

Refresh the status and metadata of application templates.
