# Cluster Commands

**Required Permissions:** ArgoCD cluster read/write access

## [Add Cluster](./add-cluster.md)

Add a Kubernetes cluster to ArgoCD for application deployment.

## [Refresh Clusters](./refresh-clusters.md)

Refresh the status and information of ArgoCD clusters.

## [Delete Cluster](./delete-cluster.md)

Delete a Kubernetes cluster from ArgoCD.
